# COMSOL multiphysics 6.2 Russian Locale — русская локализация COMSOL

Неофициальный проект русской локализации COMSOL Multiphysics 6.2

Цель проекта — сделать COMSOL более доступным для русскоязычных пользователей: студентов, инженеров, исследователей и всех, кто работает с численным моделированием, мультифизикой и инженерными расчётами.

Проект добавляет в COMSOL отдельный русский язык `ru_RU`, не заменяя штатные языки программы. После установки в настройках языка COMSOL должен появиться отдельный пункт Russian / Русский.

> Проект не является официальной локализацией COMSOL и не связан с COMSOL AB. Все права на COMSOL принадлежат их правообладателям.

## Статус проекта

Проект находится в разработке.

Перевод создаётся как с помощью машинного перевода, так и вручную. Поэтому в интерфейсе могут встречаться неточности, неудачные формулировки, разные варианты перевода одних и тех же терминов или непереведённые строки.

Буду рад любой помощи в переводе этой прекрасной программы.

Особенно полезны:

- исправления неточных или неестественных формулировок;
- вычитка технических терминов;
- перевод оставшихся английских строк;
- предложения по единому словарю терминов;
- сообщения об ошибках после установки;
- помощь в пополнении русской документации.

В будущем репозиторий будет пополняться русской документацией, примерами и пояснениями по работе с COMSOL.

## Зачем это нужно

COMSOL Multiphysics — мощная среда для моделирования физических процессов, инженерных задач и мультифизических систем. Но англоязычный интерфейс может быть дополнительным барьером для начинающих пользователей.

Русская локализация помогает:

- быстрее разобраться в интерфейсе;
- легче изучать базовые инструменты COMSOL;
- использовать программу в учебных целях;
- снизить порог входа для русскоязычных инженеров и студентов;
- сделать работу с COMSOL более комфортной.

## Установка

1. Закройте COMSOL.
2. Откройте PowerShell от имени администратора.
3. Перейдите в папку проекта:

```powershell
cd C:\Users\Admin\Desktop\ru\comsol-russian-locale
```

4. Запустите установщик:

```powershell
powershell -ExecutionPolicy Bypass -File .\install.ps1
```

Если COMSOL установлен не в стандартной папке:

```powershell
powershell -ExecutionPolicy Bypass -File .\install.ps1 -ComsolRoot "D:\COMSOL\COMSOL62\Multiphysics"
```

Для запуска без паузы в конце:

```powershell
powershell -ExecutionPolicy Bypass -File .\install.ps1 -NoPause
```

## Что делает установщик

Скрипт ищет COMSOL в стандартных папках `Program Files`, добавляет файлы `translations\ru_RU\*_ru_RU.properties` в архивы `com.comsol.resources_*.jar`, затем патчит `com.comsol.util_*.jar`: заменяет неиспользуемый кандидат языка `sv_SE` на `ru_RU` в списке доступных языков.

Также скрипт:

- создаёт backup изменяемых JAR-файлов;
- выставляет текущий язык в `ru_RU`;
- сбрасывает кэш Eclipse/OSGi, чтобы COMSOL перечитал ресурсы;
- создаёт локальный `install-state.json` для отката.

## Откат

Откройте PowerShell от имени администратора и выполните:

```powershell
powershell -ExecutionPolicy Bypass -File .\uninstall.ps1
```

Откат восстановит JAR-файлы и настройки из backup-файлов, записанных в `install-state.json`.

## Как помочь проекту

Если вы хотите помочь с переводом:

1. Сделайте fork репозитория.
2. Исправьте или дополните файлы перевода в папке `translations\ru_RU`.
3. Проверьте, что COMSOL запускается и русский язык отображается корректно.
4. Создайте Pull Request с кратким описанием изменений.

Также можно создать Issue, если вы нашли:

- ошибку перевода;
- непереведённую строку;
- проблему установки;
- некорректный термин;
- место, где перевод мешает понять смысл.

## Рекомендации по переводу

Желательно придерживаться единого стиля:

- переводить технические термины аккуратно, без чрезмерного упрощения;
- сохранять смысл оригинального интерфейса;
- не переводить названия модулей, методов и физических интерфейсов там, где это может запутать пользователя;
- избегать слишком длинных строк, если они могут не поместиться в интерфейс;
- по возможности сверять перевод с русскоязычной инженерной терминологией.

## Структура проекта

```text
translations\ru_RU\  русские файлы локали
install.ps1          установка русского языка ru_RU
uninstall.ps1        откат установки
README.md            инструкция и описание проекта
```

Файлы `*.jar`, backup-файлы и `install-state.json` не нужно добавлять в Git.

## Важно

Перед установкой рекомендуется сделать резервную копию важных данных и закрыть COMSOL.

Проект изменяет локальные файлы установки COMSOL. Используйте его на свой риск. Если что-то пошло не так, воспользуйтесь `uninstall.ps1` для отката изменений.
